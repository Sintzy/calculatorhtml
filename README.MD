# HTML Calculator

Uma calculadora em html.

## Instalação


Estaria o repositório para uma pasta: 
```bash
git https://github.com/sintzy/calculatorhtml
```
(usando o gitbash)

## Como usar
A pasta já está configurada, pode usar o html para seu website.

Lembre-se de colocar a pasta css e js, se não a calculadora pode não funcionar!
